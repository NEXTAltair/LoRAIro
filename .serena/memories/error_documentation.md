# エラードキュメント

このメモリは、プロジェクト内の主要な障害ポイントとその解決方法を記録するためのものです。各エラーは以下の形式で記録されます：

## エラーテンプレート

```
### [エラーID] - [エラータイトル]

**発生状況**: いつ、どのような操作で発生したか
**エラーメッセージ**: 実際のエラーメッセージ
**原因**: 根本的な原因の分析
**解決方法**: 採用した解決策
**予防策**: 今後同様のエラーを防ぐための対策
**関連コミット/PR**: 修正を含むコミットやPR（該当する場合）
```

## 記録済みエラー

現在、記録済みのエラーはありません。新しいエラーが発生し解決された場合、このメモリを更新します。

## エラー記録のガイドライン

1. 一時的なエラーではなく、再現性のある重要な問題を記録する
2. 解決策だけでなく根本原因の分析も含める
3. 同様のエラーを防ぐための予防策も記録する
4. 関連するコード変更やPRへの参照を含める
5. エラーの重要度や影響範囲も記載する
