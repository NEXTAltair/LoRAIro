設計メモ — このワイヤーフレームが言いたいことDesign memo — what these wires argue for

LoRAIro は 「LoRA学習用データセットを、品質を保証しながら作る道具」。アノテーション結果は 必ず人間が一枚ずつレビューするのではなく、機械にトリアージさせて要レビューだけ人が触るのが本設計の中心仮説。各フレームは 「次にどんな1アクションを取らせるか」 を1つに絞っている。 LoRAIro is "a tool for building LoRA training sets while keeping quality auditable". Core hypothesis: let the machine triage, let humans review only what's flagged — not one-image-at-a-time approval. Every frame is built around a single next action.
